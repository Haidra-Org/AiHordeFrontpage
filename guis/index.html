AI Horde GUIs and Tools

Explore our collection of user interfaces and developer tools for AI Horde. From web-based clients to desktop applications, mobile apps, SDKs, plugins, and bots - find everything you need to work with AI Horde.

New to AI Horde? Here's what you'll find:

AI Horde is a community-powered network where volunteers share their computers to generate AI images and text for free.

GUIs & Frontends
Web, mobile, and desktop apps for creating AI images and text
Workers

Developer Tools
SDKs, libraries, and plugins for integration

Most users start with a GUI/Frontend. We recommend ArtBot for beginners - it's web-based, so there's nothing to install. Use the filters below to find the perfect tool for your needs. Have questions? Check our FAQ.

Become a Worker Donate KudosThe priority system of the AI Horde. Workers earn kudos for processing jobs, and users spend kudos for higher priority in the queue. Kudos never expire and cannot be sold (but they can be gifted to other users); they represent your contribution to the community.

Image 10Graphical frontends for generating images and text. These are the easiest way to use AI Horde: just open one in your browser and start creating.

Artbot ★ Recommended

One of the most popular web-based clients. No registration required. Features include an image gallery, pending image tracking, model details browser, worker management, and a JSON payload tool. Has been used to generate over 39 million images.

Image Web Open Source
Visit GitHub

Lucid Creations ★ Recommended

A cross-platform client by the founder of the AI Horde project. Supports all available features of the API, including advanced options like LoRAs, ControlNet, and upscalers. Runs natively on Windows, Linux, and macOS with no GPU required, and also has a web version.

Image Desktop, Web Open Source
Get Lucid Creations GitHub

AI Scribbles ★ Easy to Use

A beginner-friendly web image generator powered by AI Horde, featuring handmade custom styles, prompt enhancement, and optional Discord Webhook integration to post generated images directly to a channel.

HordeNG ★ Recommended

A progressive web app for AI Horde image generation, designed to be approachable for beginners while exposing the full feature set for advanced users. Supports LoRAs, textual inversions, ControlNet, face-fixers, upscalers, HiRes Fix, and QR code generation.

Image Web Open Source
Visit GitHub

Text 5Graphical frontends for generating images and text. These are the easiest way to use AI Horde: just open one in your browser and start creating.

KoboldAI Lite ★ Recommended

A web-based frontend for LLM APIs, including AI Horde. Offers multiple UI themes (Classic, Aesthetic, Corpo), instruct mode, quick-start scenarios, save/load support, and simultaneous access to many AI Horde models.

Text Web Open Source
Visit GitHub

SillyTavern ★ Recommended

A powerful locally-installed LLM frontend supporting AI Horde, KoboldAI/CPP, NovelAI, OpenAI, Claude, and many more APIs. Features include Visual Novel Mode, image generation via Automatic1111/ComfyUI, TTS, WorldInfo (lorebooks), mobile-friendly layout, auto-translate, and a rich extension ecosystem. Over 300 contributors and 3+ years of active development.

Text Desktop Open Source
Get SillyTavern GitHub

Inneal ★ Recommended

A native iOS and iPadOS LLM roleplay chatbot client for the AI Horde. Import Tavern character cards, enjoy multi-character chat, personas, message swiping, and an 'Auto Mode' for easy setup. Syncs chats and characters across devices via CloudKit.

Text iOS Open Source
Get Inneal GitHub

AgnAIstic

Open source AI character chat platform, publicly hosted at agnai.chat or self-hostable. Supports AI Horde and many other AI services, with features including image generation, long-term memory/embeddings, character expressions, memory books, stop generation, and chat graphs.

Text Web Open Source
Visit GitHub

TavernAI

An adventure atmospheric chat client supporting multiple AI backends including KoboldAI, NovelAI, Pygmalion, and OpenAI ChatGPT. Can be downloaded locally or tried online via Google Colab. The predecessor project that inspired SillyTavern.

Text Desktop Open Source
Visit GitHub
