Quickstart Everything you need to start generating. You can use the anonymous API key right away, with no registration required. Registering gives you kudos (priority credits) for faster results.

First register an account which will generate an API key. Store that key somewhere safe - treat it like a password.

  • If you do not want to register, you can use 0000000000 (10 zeros) as an API key to connect anonymously. However, anonymous accounts have the lowest priority when there are too many concurrent requests!
  • To increase your priority you will need a unique API key and then to increase your KudosThe priority system of the AI Horde. Workers earn kudos for processing jobs, and users spend kudos for higher priority in the queue. Kudos never expire and cannot be sold (but they can be gifted to other users); they represent your contribution to the community.
